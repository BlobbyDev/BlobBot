Features

Anti Toxicity

Advanced anti-toxicity system that eases the lives of moderators and helps to lower the level of toxicity in the server.

Ticket System

A flexible ticket system that facilitates support to members through support team.

Message Logging

An advnaced message-logging system which helps your moderators to monitor server messages.

Autorole System

A simple auto-role system that assigns new members a role.

Join & Leave Logs

Helps the server to greet new users and notify who have left the server.

Leveling System

A simple leveling system that motivates members to be active in the server.

Global Economy

A fun economy where you can purchase items, sell items, and gamble with our global currency.

Frequently Asked Questions

The bot runs on Discord's slash commands. Run "/help" to view all the commands within the bot in the bot.
The "tolerance" option specifies the percentage of toxicity the server can handle. For instance, if you set it to 70, the bot will delete messages that exceed the 70% tolerance and log the removed messages to your logging channel. Run "/analyze-toxicity" to verify the calculation of toxicity %.
Unfortunately, the Auto Role system only allows you to add one role. To change the role, run "/autorole-disable" and then "/autorole-configure" to reconfigure it.
Run "/report-bug" and then explain the bug in detail.
